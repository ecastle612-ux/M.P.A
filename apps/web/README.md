# Web App Foundation

Next.js App Router scaffold for Phase 2 foundation.

## Included

- Authentication framework (login/logout/session/protected route guard)
- Role-aware shell framework
- Canopy tokenized theme provider integration
- PWA manifest + service-worker registration scaffold
- Responsive app shell placeholders

## Excluded (by design)

- Property, tenant, lease, marketplace, payment, maintenance, AI, and document features
- Business workflows and domain tables
