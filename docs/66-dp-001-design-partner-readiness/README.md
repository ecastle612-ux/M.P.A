# DP-001 — Design Partner Readiness Sprint

**Status:** Complete  
**Date:** 2026-07-19  
**Parent:** EP-004 · OPS-003 Launch Readiness Certification

## Purpose

Close the last major Design Partner readiness gaps so beta companies never encounter unfinished experiences. No new platform features. No architecture redesign. No changes to completed workflows (Accounting, Maintenance, Timeline, Resident workflows, Reporting engine, Facility architecture, Command Center, Operations Center).

## P0 scope (implement only)

| ID | Surface | Outcome |
|----|---------|---------|
| P0-1 | Settings → Organization / Team | Permanent org info, invite staff, members, roles, permissions summary, pending invitations, deactivate |
| P0-2 | Settings → Integrations | Provider Status Center (Connected / Sandbox / Disconnected + guidance) |
| P0-3 | Portal availability | Gate unfinished Owner / Manager portals; no dead-end navigation |
| P0-4 | Settings → Documents | PM Document Vault browser over existing vault (search, filter, preview, download) |
