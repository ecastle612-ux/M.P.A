# MX-001 — Customer Migration Center Foundation

**Status:** Implemented (scoped foundation)  
**Objective:** Enable orgs to migrate from legacy PM software via CSV/Excel/ZIP/folder imports with mapping, review queue, duplicates, rollback, and onboarding wizard UX.

## Out of scope (foundation)

OCR, AI matching, direct API connectors, accounting/payment/email migration, native desktop uploader.

## PRR IDs

MHF-002, MHF-003, MHF-005, MHF-012, MHF-015, FEH-1203 (onboarding/migration tooling)

## Access

`migration:*` capabilities — property_manager + property_owner only (hidden from tenant/vendor/resident planes).
