# 08 — Mobile Experience

**Package:** DPX-003  
**Status:** Draft — awaiting Approve  
**Viewport standard:** 390×844 (one-hand)

---

## Requirements

- Comfortable one-hand use  
- Reduced scrolling vs baseline  
- Thumb-reachable primary actions  
- Floating controls never block content  
- No clipped UI  
- No keyboard overlap traps on forms  
- No accidental control overlap  

## Method

1. Capture scroll height / screenfuls before  
2. Apply hierarchy + disclose from [02](./02-cognitive-load-audit.md)  
3. Re-measure; file remaining MOB friction  

Align with UX-009 mobile standard; do not expand nav.
