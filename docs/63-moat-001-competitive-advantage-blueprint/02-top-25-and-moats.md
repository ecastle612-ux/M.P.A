# MOAT-001 — Top 25 Competitive Advantages & Defensible Moats

**Status:** Draft — Ready for Approval

Copyability:

| Tier | Meaning |
| --- | --- |
| **Easy** | UI pattern or feature competitors can ship in a release cycle |
| **Hard** | Requires architecture + workflow rewrite + cultural product discipline |
| **Nearly impossible** | Compounded data, habits, trust, and multi-year OS consistency |

| 16 | **Offline inspections** | Hard | Sync conflict UX + field reliability (CA-007) |

| 18 | **Bulk ops with safe enums** | Easy/Medium | Tables are easy; capability-safe transitions are medium |
| 19 | **Interrupted-user design** (3-min recovery) | Nearly impossible* | *As a *felt* culture across the whole product over years |
| 20 | **Org-scoped AI learning from accept/dismiss** | Nearly impossible | Compounding per-tenant memory; not cross-tenant leakage |

## 5. Defensible moats (compounding layers)

Moats stack. Single features do not.

```
Layer 4 — Compounding memory (AI feedback, automations, team habits, KB)
Layer 3 — Trust rails (explainability, human gates, audit, RLS)
Layer 2 — OS interaction model (Today’s Work, Palette, Inspector, Quick Add)
Layer 1 — Workflow grammar (lifecycle jobs, domain events, operational graph)
Layer 0 — Adoption gates (payments, screening, e-sign, delivery) — necessary, not sufficient
```

| Moat | Defensibility | Failure mode if neglected |
| --- | --- | --- |
| **Ops OS loop** | Hard | Becomes DoorLoop-with-extra-steps |
| **Explainable supervised AI** | Hard | Race to unsafe autonomy; trust collapse |
| **Lifecycle grammar** | Hard | Dual CRUD paths return (DX-003 regression) |
| **Migration trust** | Hard | Dual-system forever; churn at week 2 |
| **Comms graph + QR** | Hard | Back to texting outside the system |
| **Org learning** | Nearly impossible | Generic AI; no retention wedge |
| **Accounting depth** | Easy for giants | Do not bet Year-1 identity here |

### Why leave → why stay (closed loop)

```
Leave competitor: training tax + slow interrupt recovery
     ↓
Adopt M.P.A.: Today’s Work + guided jobs + Quick Add
     ↓
Stay: team habits + AI weights + automation rules + comms history
     + migration already paid + owners/residents enrolled
```
