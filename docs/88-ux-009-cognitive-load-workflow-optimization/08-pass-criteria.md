## Hard PASS (all required)

| Gate | Criterion |
| --- | --- |
| G1 | Primary actions immediately visible (header and/or toolbelt) on major entities (Amendment A) |
| G2 | Mobile scrolling substantially reduced (≥40% avg on audited detail pages, or documented exception with disclosure) (G) |
| G3 | AI available everywhere via floating **copilot** without dominating page content (D) |
| G4 | Search is a primary navigation method; replaces unnecessary nav for scripted tasks (C, E) |
| G5 | One-glance headers: where / attention / immediate actions in &lt;3s (B) |
| G6 | Common tasks need less thinking / fewer taps / less scrolling; friction removed (I) |
| G7 | No feature removal; no permission/API/schema changes; a11y not regressed |
| G8 | Design Partner first-time-PM test: no “Where is that?” / “What does this do?” / “Why is this here?” (J) |
| G9 | Interface feels calm; users spend more time managing properties than navigating software |
