# BR-002 — Visual Brand Certification (Human Standard)

**Initiative ID:** BR-002  
**Status:** Design - Approved · Document - Approved · **ADR-022 Accepted** · Implement unlocked  
**Type:** Perception-based brand quality + purpose-optimized presentation  
**Depends on:** BR-001 / ADR-021 (Accepted)  
**Amendment:** [BR-002A — “Looks Great” Standard](./07-amendment-br-002a.md)

---

## Decision intent

Certify the **visual experience**, not component wiring.

> If this were the first screen a property manager ever saw, would I proudly demo it to a paying customer?

**Brand recognition takes priority over logo fidelity.**

## Success criterion

After BR-002 is implemented and human-certified, branding is a **solved problem**. Future work is AI, workflows, and Design Partner feedback — not logo visibility.
