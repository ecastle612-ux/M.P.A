## Objective

Act as a property manager running **250 units** from 08:00–17:00 in M.P.A.  
Find every unnecessary click, duplicate path, dead end, and training burden — then recommend **workflow eliminations** that save time, reduce clicks, reduce training, and reduce mistakes.

**This package does not authorize implementation.**

## Rules (binding for implementers after Approve)

1. Prefer improving existing workflows — **do not invent new modules**.
2. Every approved change must save time, reduce clicks, reduce training, and reduce mistakes.
3. Inspections productization is **out of scope** for DX-003 implementation slices (documented as a dead-end only).
4. Material scope changes restart Design → Document → Approve.
