## Objective

Every common property-management task must be completable in **≤ 5 minutes without training**.  
M.P.A. must feel like an **operating system for portfolio work**, not a collection of pages.

**No application code in this package.**

## Binding rules (post-Approve)

1. Prefer extending Operations Center, Command Center, lists, and guided wizards — **do not invent parallel apps**.
2. Every approved change must pass the **5-Minute Rule** test for a common job.
3. AI “Next Best Action” recommends; humans confirm — no silent mutations of money, leases, or access.
4. Material scope changes restart Design → Document → Approve.
