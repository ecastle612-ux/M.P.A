## Objective

Make `/master-admin` the permanent **CEO / Platform Administrator headquarters** for operating M.P.A.

This is **Mission Control** — not another dashboard. On login, actionable reality is surfaced first; the operator should not have to decide where to go.

---

## Four levels (home)

1. **Immediate Attention** — critical items always visible  
2. **Business Snapshot** — live KPIs  
3. **Operational Workspaces** — Platform · Customers · Operations · Support · Sales · Development · Analytics  
4. **Quick Actions** — one-click constant jobs  

Plus **universal Global Search**. Executive AI Brief = Slice C (not chat).
