# 04 — AI Copilot in the Workflow

**Package:** DPX-002  
**Amendment:** D  
**Status:** Approved  
**Rule:** AI is an operational partner. It **reduces navigation** — it must not increase interaction.

---

## Required assists (certified path)

| Context | Assist |
| --- | --- |
| Dashboard | What should I work on first today? |
| Resident | Summarize account · Draft message · Explain payment history |
| Maintenance / WO | Suggest next step · Draft vendor message |
| Property | Outstanding issues · Vacancy recommendations |

## Integration rules

1. Suggestions match page context store for the current entity.  
2. Prefer deep links into existing routes over prose-only dead ends.  
3. Copilot accelerates toolbelt / next actions — does not replace them as a second workflow.  
4. Path remains completable with AI closed (permissions / offline).  
5. Momentum Rule (E): AI must not add unnecessary steps or dialogs.
