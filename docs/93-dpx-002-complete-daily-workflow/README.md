## Objective

Complete the Property → Resident → Lease → Payment → Maintenance → Vendor → Communication path until it feels production-ready and differentiates M.P.A. from AppFolio, Buildium, Rent Manager, and peers — through **operator experience**, not feature count.

> No other workflow should be implemented to a lower standard.

## Certified path

1. Review today’s priorities  
2. Locate a property  
3. Open a resident  
4. View lease status  
5. Collect payment information  
6. Review maintenance  
7. Assign a vendor  
8. Communicate with the resident  
9. Notify the owner if necessary  
10. Return to dashboard
