# 21 — Experience Architecture

**Phase:** 1.6 — Experience Architecture  
**Status:** Approved (v1.0)  
**Relationship:** Complements **06 Canopy** (how it looks) and **07 UX Principles** (how it works). This document defines **how it feels**.

---

## Mandate

People will remember M.P.A. because of how it makes them feel.

Every interaction should:

| Outcome | Meaning |
|---------|---------|
| Reduce stress | Lower cognitive load; remove “what am I missing?” |
| Increase confidence | User knows the next right action |
| Reduce uncertainty | Status, ownership, and timing are always clear |
| Increase productivity | Fewer clicks, less hunting, less re-entry |
| Create trust | Honest urgency, sourced AI, no dark patterns |

The software should feel like an **experienced operations manager working alongside the user** — calm, prepared, specific, and never theatrical.

## Experience North Star

> M.P.A. makes property operations feel **manageable** — even when the portfolio is large and the day is hard.

| Role | Emotional promise |
|------|-------------------|
| Property Manager | “I am ahead — I know what matters.” |
| Tenant | “Living here is simple — I am taken care of.” |
| Property Owner | “My investment is visible and protected.” |
| Vendor | “Good work is easy to do and easy to get paid for.” |

One product language (Canopy). Four emotional journeys.

---

## The Metaphor

M.P.A. is not a filing cabinet, a spreadsheet, or a chat bot.

It is a **calm co-pilot for operations**:

- Surfaces what matters
- Explains what happens next
- Remembers context so the user does not have to
- Steps back when the user is in flow
- Never shames failure; always teaches recovery

## The M.P.A. Experience Test

Before approving this phase, answer for a first-time user:

| Question | Target |
|----------|--------|
| Would they immediately trust it? | Yes — clarity, honesty, calm urgency |
| Would they understand it? | Yes — zero-training goal |
| Would they enjoy it? | Yes — progress feels rewarding; chrome stays out of the way |
| Would they recommend it? | Yes — “it reduced my stress” |
| Would they feel less stressed than before opening it? | **Required** — if opening M.P.A. raises anxiety, redesign |
