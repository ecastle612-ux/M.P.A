# 01 — Process (permanent)

A bugfix is incomplete until:

1. Implemented  
2. Committed  
3. Pushed  
4. Deployed  
5. Deployment Verified (commit + bundle on the URL under test)  
6. Live workflow retested on that deployment  
7. User Verified (or Cursor-verified with evidence)

Never mark PASS from TypeScript, unit tests, or code review alone.
